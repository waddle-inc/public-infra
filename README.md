# インフラリポジトリ - Waddle Inc.

本番 VPS 上のサービスを Docker Compose で運用するためのインフラ定義です（アプリケーションのソースは含みません）。本番 VPS で `deploy.sh` を実行すると、アプリソースの取得からビルド・起動までまとめて行います。
